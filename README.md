## Using Frameworks

### frontend

* Vue.js
* Webpack
* ES6

### backend

* Python 3.5.2
* Flask
* SQLAlchemy
* FlaskTesting  
* selenium

## How to start

Add this to your shell config file (ex: .bashrc)

```sh
HOW_WAS_YOUR_DAY_ENV=local flask run
```
