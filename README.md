## Using Frameworks

### frontend

* Vue.js
* Webpack
* ES6

### backend

* Python 3.5.2
* Flask
* SQLAlchemy
* Pytest

## How to start

Add this to your shell config file (ex: `.bashrc`)

```sh
FLASK_APP=wsgi.py
HOW_WAS_YOUR_DAY_ENV=local
```

Run this command in your python-virtualenv

```sh
flask run
```
